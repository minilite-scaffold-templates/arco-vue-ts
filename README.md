# ARCO-VUE-TS TEMPLATE

## 基于 Arco-ui, Vue3.2, Vite2+, TypeScript, Windicss 搭建 PC 端前端模板项目

```sh

  # 安装依赖
  yarn install

  # 开发环境下运行
  yarn dev

  # 打包生产环境
  yarn build

  # 在本地启动nodejs服务器运行打包后的项目
  yarn gzip

```
